Biology Animation Library

A searchable GitHub Pages landing page for interactive class animations. The library was rebuilt from a ZIP file whose folders are used as category sections.

Animation Prompted by Jerred Seveyka for his biology class.

To add a new animation later, put it in the appropriate folder inside animations/, then add one matching entry to animations.js. GitHub Pages cannot automatically scan folders after upload, so the list file needs to be updated when new files are added.
